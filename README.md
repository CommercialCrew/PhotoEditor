# UI3 Photo-Editer

Windows 11 기준으로 파일 탐색기에서 찾아갈 수 있는 '갤러리' 내의 사진 파일들을 인식해서 리스트업하고, 사진을 선택해서 간단하게 필터를 입힐 수 있는 프로그램입니다.
